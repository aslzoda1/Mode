### ✨ Exclusive Features

* 🎯 **Smart Persistence:** Tanlangan rejim (Light/Dark) `SharedPreferences` yordamida doimiy eslab qolinadi.
* 🌈 **Fluid Transitions:** Mavzular o'rtasidagi o'tishlar `ValueAnimator` orqali ko'zga yoqimli va silliq amalga oshiriladi.
* 💾 **User Profile:** Ismingizni saqlash, tahrirlash va o'chirish imkoniyati.
* 🎨 **Modern Glassmorphism:** Zamonaviy ranglar palitrasi va yumshoq chekkali (rounded) dizayn elementlari.
* ⚡ **Optimized Performance:** Minimal resurs sarfi bilan maksimal tezlik.

---

### 🛠 Technical Blueprint

> **Architecture:** Modern Imperative UI  
> **Backend Logic:** Java SE  
> **Styling:** Custom XML Vector Drawables  
> **Data Layer:** Lightweight Key-Value Storage (SharedPreferences)
